Installation

  1. Download the latest ISO file
  2. Plug in a thumb drive that is AT LEAST 8GB
  3. Using a third-party software (E. g.: Rufus) create a bootable USB drive
  4. After the USB drive is ready, plug it into a computer
  5. In the BIOS/UEFI change the boot-order, so the computer boots from the USB drive
  6. After the computer has restarted, continue with the installation. The computer will restart several times
IMPORTANT: DO NOT remove the installation media (USB drive) until the computer has booted into the main interface of NeuraSys

Commands

In the examples, the "->" means you have to press "ENTER" (To give further parameters)

help

Gives you all the available commands with descriptions

Further parameters:

None

Example:

help

time

Shows the current time in your region

Further parameters:

None

Example:

time

create

Creates a file in your current directory

Further parameters:

Name of the file (with the extension)

File's content (Use \n to create new lines)

Example:

create -> start.bat -> @echo off\nexplorer.exe

delete

Deletes the specified file in your current directory

Further parameters:

Name of the file (with the extension)

Example:

delete -> start.bat

list

Lists the content of the current directory

Further parameters:

None

Example:

list

read

Reads the specified file's content

Further parameters:

Name of the file (with the extension)

Example:

read -> start.bat

rename

Renames a file

Further parameters:

Current name of the file (with the extension)

New name of the file (with the extension)

Example:

rename -> start.bat -> stop.bat
